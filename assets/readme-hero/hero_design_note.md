# Research Investigation Workflow README Hero Design Note

## Project Summary

Research Investigation Workflow is a Codex skill that coordinates TraceGuard, LogicGuard, FlowGuard, and a local run-history ledger for deep multi-round investigations.

## Target Users

AI-assisted researchers, analysts, investigators, technical writers, and maintainers who need sourced reports with clear evidence boundaries.

## Core Problem

Investigation runs often mix messy evidence, source support, report prose, process state, and reusable run lessons. This skill keeps those layers separate while still coordinating them into one workflow.

## Core Workflow

Public sources, local notes, and evidence cards move through TraceGuard lead reconstruction, LogicGuard claim and citation structuring, FlowGuard closure checks, and a History Ledger feedback loop before producing a report.
